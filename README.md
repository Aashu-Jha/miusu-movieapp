# miusu

A new Flutter movie project can be used to see the upcoming, popular movies. Make your playlists.

## Functionalities used - 
  - Cubit/ Bloc State management
  - Hive 
  - wiredash (Feedback Api)
  - Tmdb Api
  - Tmdb Authentication
  - Youtube Player Flutter
  - Splash Screen
  and many more...
